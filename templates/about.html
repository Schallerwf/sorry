Making of Ultimate Sorry

Sorry is a fantastic board game. Over the years it has become my go to game for friend and family gatherings. I often wondered how much of Sorry was luck vs skill. I built a Sorry simulator in python that can play thousands of games in a few milliseconds. The program can test various stategies, rule changes, and view the impact on the game. Then I built a web server to allow a single board position to be analyzed. If you are ever wondering what might be the best move in a Sorry game the move analyzer will help (or confuse) you.

You can provide feedback or see my other projects on my website WilliamSchaller.com
